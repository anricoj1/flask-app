# Dependencies
  - pip3 install Flask-Mail
  - pip3 install WTForms-Components
  - pip3 install flask
  - pip3 install PyMySQL
  - pip3 install flask-WTF
  - pip3 install passlib
  - pip3 install flask-bootstrap

# To Run
  - python3 app.py
  - Make Sure you have MySQL and mysql-server installed
